# Veil

**Trade privately, settle publicly.**

Private DEX on [MagicBlock](https://www.magicblock.xyz/) Private Ephemeral Rollups — shield SPL tokens, swap without mempool exposure, unshield to Solana.

Built for [Solana Blitz v5](https://hackathon.magicblock.app/).

## Features

- **Shield** — deposit canonical SPL tokens (SOL, USDC) into a Private ER
- **Swap** — private swaps with live price quotes
- **Unshield** — withdraw SPL back to your Solana wallet

## Stack

Next.js · TypeScript · Tailwind · shadcn/ui · Solana Wallet Adapter · MagicBlock Private Payments API

## Getting started

```bash
bun install
bun dev
```

Open [http://localhost:3000](http://localhost:3000).
